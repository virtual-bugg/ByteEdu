Interpolation with Curly Braces

String interpolation with curly braces is a way of embedding variables into strings. To use string interpolation with curly braces, simply enclose the variable name in curly braces within a string.

    $name = "John Doe";
    $age = 30;
  
    echo "Hello, {$name}! You are {$age} years old.";

The above code will print the following output:

  Hello, John Doe! You are 30 years old.

Characters and String Indexes

Characters in a string can be accessed using their index. The first character in a string has an index of 0. The second character has an index of 1, and so on.

    $name = "John Doe";
  
    echo $name[0]; // Prints "J"
    echo $name[3]; // Prints "h"

String Operators

PHP provides a number of string operators. Some common string operators include:

  • . (concatenation operator)
  • == (comparison operator)
  • === (identical comparison operator)
  • !== (not identical comparison operator)
    $string1 = "Hello, ";
    $string2 = "John Doe!";
  
    echo $string1 . $string2; // Prints "Hello, John Doe!"
    echo $string1 == $string2; // Prints false
    echo $string1 === $string2; // Prints false

Heredoc

Heredoc is a way of creating multi-line strings. To use heredoc, simply enclose the string in a pair of "<<<" and ">>>" symbols. The heredoc syntax can be used to create strings that contain newlines and other special characters.

    $name = "John Doe";
  
    $string = <<<EOT
    Hello, $name!
  
    How are you today?
    EOT;
  
    echo $string;

The above code will print the following output:

  Hello, John Doe!

  How are you today?

String Functions

PHP provides a number of built-in string functions. Some common string functions include:

  • strlen() - Returns the length of a string
  • strtoupper() - Converts a string to uppercase
  • strtolower() - Converts a string to lowercase
  • str_replace() - Replaces all occurrences of a string with another string
  • substr() - Returns a substring from a string
    $string = "Hello, world!";
  
    echo strlen($string); // Prints 12
    echo strtoupper($string); // Prints HELLO, WORLD!
    echo strtolower($string); // prints hello, world!
    echo str_replace("world", "John Doe", $string); // Prints Hello, John Doe!
    echo substr($string, 7); // Prints world!

Formatting Strings

Strings can be formatted using the `printf()` function. The `printf()` function takes a format string as its first argument. The format string can contain placeholders that are replaced with values.

    $name = "John Doe";
    $age = 30;
  
    printf("Hello, %s! You are %d years old.", $name, $age);

The above code will print the following output:

  Hello, John Doe! You are 30 years old.
