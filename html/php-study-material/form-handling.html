Working with HTML Form Controls and PHP

HTML form controls can be used to collect user input. The user input can then be accessed in PHP using super global variables.

Super Global Variables

Super global variables are variables that are available in all scopes throughout a PHP script.

The following are the super global variables that are used for form handling:

  • $_GET
  • $_POST
  • $_REQUEST

Super Global Array

The super global array is an associative array that contains the values of all form controls.

The following are the keys in the super global array:

  • name
  • value
  • type

Importing User Input

User input can be imported into PHP using the `$_GET`, `$_POST`, or `$_REQUEST` super global variables.

The `$_GET` super global variable is used to import user input that is submitted using the `GET` method.

The `$_POST` super global variable is used to import user input that is submitted using the `POST` method.

The `$_REQUEST` super global variable is used to import user input that is submitted using either the `GET` or `POST` method.

Accessing User Input

User input can be accessed using the following syntax:

      $_GET['name']
      $_POST['name']
      $_REQUEST['name']

For example, if the following form is submitted:

    <form method="GET" action="">
    <input type="text" name="name" value="John Doe">
    <input type="submit" value="Submit">
    </form>

The user input can be accessed using the following code:

    $name = $_GET['name'];
  
    echo $name; // Prints John Doe
