Creating Arrays

Arrays can be created using the `array()` constructor or the `[]` shortcut syntax.

    // Using the array() constructor
    $array1 = array(1, 2, 3);
  
    // Using the [] shortcut syntax
    $array2 = [1, 2, 3];

Accessing Array Elements

Array elements can be accessed using their index. The first element in an array has an index of 0. The second element has an index of 1, and so on.

    $array = [1, 2, 3];
  
    echo $array[0]; // Prints 1
    echo $array[1]; // Prints 2
    echo $array[2]; // Prints 3

Multidimensional Arrays

Multidimensional arrays are arrays that contain other arrays. Multidimensional arrays can be created using the `array()` constructor or the `[]` shortcut syntax.

    // Using the array() constructor
    $array1 = array(
      array(1, 2, 3),
      array(4, 5, 6),
      array(7, 8, 9)
    );
  
    // Using the [] shortcut syntax
    $array2 = [
      [1, 2, 3],
      [4, 5, 6],
      [7, 8, 9]
    ];

Inspecting Arrays

The `print_r()` function can be used to print the contents of an array.

    $array = [1, 2, 3];
  
    print_r($array);

The above code will print the following output:

    Array
    (
      [0] => 1
      [1] => 2
      [2] => 3
    )

Deleting from Arrays

The `unset()` function can be used to delete elements from an array.

    $array = [1, 2, 3];
  
    unset($array[1]);
  
    print_r($array);

The above code will print the following output:

  Array
  (
    [0] => 1
    [2] => 3
  )

Iterating with each() and foreach()

The `each()` and `foreach()` statements can be used to iterate over the elements in an array.

    // Using the each() statement
    $array = [1, 2, 3];
  
    while ($element = each($array)) {
      echo $element['key'] . " => " . $element['value'] . "\n";
    }
  
    // Using the foreach() statement
    foreach ($array as $key => $value) {
      echo $key . " => " . $value . "\n";
    }

The above code will print the following output:

  0 => 1
  1 => 2
  2 => 3

Iterative functions: current(), next(), prev(), reset(), end()

The `current()`, `next()`, `prev()`, `reset()`, and `end()` functions can be used to control the iteration of an array.

    $array = [1, 2, 3];
  
    echo current($array); // Prints 1
    next($array);
    echo current($array); // Prints 2
    prev($array);
    echo current($array); // Prints 1
    reset($array);
    echo current($array); // Prints 1
    end($array);
    echo current($array); // Prints 3
