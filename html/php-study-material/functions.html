Function Definition

A function is a block of code that can be reused multiple times. Functions are defined using the following syntax:

    function function_name($parameter1, $parameter2, ...) {
      // Code to execute
    }

The `function_name` is the name of the function. The `parameters` are variables that are passed to the function. The `code to execute` is the code that is executed when the function is called.

Creating and Invoking User-Defined Functions

To create a user-defined function, simply define the function using the syntax above. To invoke a user-defined function, simply use the function name followed by parentheses.

    function add_numbers($number1, $number2) {
      return $number1 + $number2;
    }
  
    $result = add_numbers(1, 2);

In the above example, the `add_numbers()` function is defined. The function takes two parameters, `$number1` and `$number2`. The function returns the sum of the two parameters. The `$result` variable is then assigned the value of the `add_numbers()` function.

Formal Parameters versus Actual Parameters

Formal parameters are the parameters that are defined in the function definition. Actual parameters are the values that are passed to the function when it is invoked.

    function add_numbers($number1, $number2) {
      return $number1 + $number2;
    }
  
    add_numbers(1, 2);

In the above example, the formal parameters are `$number1` and `$number2`. The actual parameters are `1` and `2`.

Function and Variable Scope

Function scope is the set of variables that are visible to a function. Variable scope is the set of variables that are visible to a particular piece of code.

    $global_variable = 1;
  
    function my_function() {
      $local_variable = 2;
  
      echo $global_variable; // Prints 1
      echo $local_variable; // Prints 2
    }
  
    my_function();
  
    echo $global_variable; // Prints 1
    echo $local_variable; // Prints undefined

In the above example, the `$global_variable` variable is a global variable. The `$local_variable` variable is a local variable. The `global_variable` variable is visible to both the `my_function()` function and the global scope. The `local_variable` variable is only visible to the `my_function()` function.

Recursion

Recursion is a technique where a function calls itself. Recursive functions can be used to solve problems that can be broken down into smaller subproblems.

    function factorial($number) {
      if ($number == 0) {
        return 1;
      } else {
        return $number * factorial($number - 1);
      }
    }
  
    $factorial_of_5 = factorial(5);

In the above example, the `factorial()` function is a recursive function. The function calculates the factorial of a number. The factorial of a number is the product of all the positive integers less than or equal to the number.

Library Functions

PHP provides a number of built-in library functions. Library functions are functions that are provided by the PHP language. Some common library functions include:

  • strlen() - Returns the length of a string
  • strtoupper() - Converts a string to uppercase
  • strtolower() - Converts a string to lowercase
  • rand() - Generates a random number
  • sqrt() - Calculates the square root of a number

Variable-length Argument Lists

Variable-length argument lists are functions that can take any number of arguments. Variable-length argument lists are defined using the `...` operator.

    function sum_numbers(...) {
      $sum = 0;
  
      foreach ($arguments as $argument) {
        $sum += $argument;
      }
  
      return $sum;
    }
  
    $result = sum_numbers(1, 2, 3, 4, 5);

Default Parameter Values

Default parameter values are values that are assigned to parameters if no value is passed to the function. Default parameter values are defined using the `=` operator.

    function say_hello($name = "John Doe") {
      echo "Hello, $name!";
    }
  
    say_hello();

Anonymous Functions

Anonymous functions are functions that do not have a name. Anonymous functions are defined using the `fn` keyword.

    $fn = fn($number) => $number * 2;
  
    $result = $fn(2);
